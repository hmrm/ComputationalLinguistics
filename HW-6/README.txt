Calculates the string distance using A*.
Expected usage: python stringdistance.py string1 string2

Example results:

$ python stringdistance.py thenameofthegame theresmyname
String Edit Distance: 7.0
Matching:
thenameofthegame
ther  e smy name

$ python stringdistance.py ninakushukuru unamshukuru
String Edit Distance: 3.1
Matching:
ninakushukuru
 unam shukuru
